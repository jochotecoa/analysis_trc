# TRC Analysis Project

## Description

This project contains a series of R scripts for the analysis of transcriptomics (TRC) and proteomics data. The analyses include differential expression, correlation analysis, and pathway analysis to investigate the relationship between different omics layers.

## Key Improvements

- **Centralized Configuration:** Project-wide variables and paths are now in `config.R`.
- **Centralized Utilities:** Shared functions are consolidated in `utils.R`.
- **Decluttered Workspace:** Older versions of scripts have been moved to the `archive/` directory.
- **Improved Maintainability:** Main scripts have been refactored to use `config.R` and `utils.R`.

## Project Structure

- `config.R`: Centralized configuration and path variables.
- `utils.R`: Centralized utility functions used across the project.
- `01_DESeq2_RNA_Seq.R`: Differential Expression Analysis using DESeq2.
- `02_Limma_DEG_Analysis.R` (in `DEGs/`): Differential Expression Analysis using Limma.
- `03_Correlation_TPM_vs_Protein.R`: Basic correlation between TPM and protein expression.
- `04_Correlation_TRC_vs_Protein.R`: Basic correlation between TRC and protein expression.
- `05_Compare_Correlations.R`: Comparison of TPM and TRC correlations.
- `06_Correlation_With_Shifts.R`: TRC-protein correlation analysis with time-shifts enabled.
- `07_Time_Shift_Analysis.R`: Analysis of protein half-lives vs observed time-shifts.
- `archive/`: Historical versions of scripts.
- `results/`: Output directory for analysis results (generated by scripts).

## Analysis Pipeline

The project is organized into a sequential pipeline:

1.  **Differential Expression:**
    - `01_DESeq2_RNA_Seq.R`
    - `02_Limma_DEG_Analysis.R`
2.  **Correlation Analysis (Baseline):**
    - `03_Correlation_TPM_vs_Protein.R`
    - `04_Correlation_TRC_vs_Protein.R`
3.  **Comparison:**
    - `05_Compare_Correlations.R`
4.  **Advanced Analysis (Time-Shifts):**
    - `06_Correlation_With_Shifts.R`
    - `07_Time_Shift_Analysis.R`

## How to Run

1.  Open the `analysis_trc.Rproj` file in RStudio.
2.  Review and update `config.R` to ensure all data paths are correct for your environment.
3.  Run scripts sequentially (01 to 07) to perform the full analysis. Scripts 03-07 will output results into the `results/` directory.
